OBCOM DesktopFX

El producto OBCOM DesktopFX permite desarrollar soluciones RIA (Rich Internet Applications) utilizando la tecnologías Java y JavaFX de Oracle, versión 8 o superior.

Esta página permite descargar e instalar OBCOM DesktopFX. Una vez instalada, la aplicación se ejecuta haciendo clic en un ícono colocado en el escritorio. Si hay nuevas versiones disponibles, éstas se descargarán e instalarán automáticamente antes de ejecutar la aplicación. No es necesario volver a usar esta página.
